Sebby's API Docs

User database

Read

https://api.sebbymortimer.co.uk/read/<user_id>
e.g. https://api.sebbymortimer.co.uk/read/12345

Returns the name of the specified user_id

Write

https://api.sebbymortimer.co.uk/write/<user_id>/<name>
e.g. https://api.sebbymortimer.co.uk/write/12345/Sebby

Writes the given user_id and name to the database
